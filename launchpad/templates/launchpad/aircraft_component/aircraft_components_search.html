Verify Aircraft Component

You can verify the authenticity of a component via this form. Depending on the manufacturer's configuration a component ID is either embossed on the part and by default it is a three word combination for e.g. lexical-safari-prognosis

Need Help? Your Aerobridge ID should be embossed on the component or your component should have a sticker / QR code on it.
