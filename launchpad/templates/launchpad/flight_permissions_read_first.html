Get started with Flight Permissions

Flight permissions must be obtained via Digital Sky, you can store them here. The following is the flow for permission artefacts:



  1. Create a Flight Plan (Upload Coordinates file as GeoJSON in Aerobridge)
  2. Create a Flight Operation: Assign a flight plan to a operation and a drone, you will have to input additional fields around the operation.
  3. As of May 2021, at this stage you have to use the Digital Sky Application to submit the flight plan
  4. You can then download a permission artefact or apply for one if you have done so.
  5. After the operation is complete the signed flight logs from the drone can be uploaded to the server.
