Get started with Flight Permissions

Flight permissions must be obtained via Digital Sky, you can store them here. The following is the flow for permission artefacts:



  1. Create a Flight Plan (Upload Coordinates file as GeoJSON in Aerobridge)
  2. Create a Flight Operation: Assign a flight plan to a operation and a drone, you will have to input additional fields around the operation.
  3. Create a Flight Permission Object, this will have the details of the operation against which the permission is requested. As of May 2021,you have to use the Digital Sky Application to submit the flight plan.
  4. Once a flight permission object is created, you can then use the "Request Flight Permission" link in the DigitalSky menu to request a permission artefact.
  5. Once the flight permission isssued, you can see the permission artefact at in the "Flight Permissions" menu in the "Plans and Operations" list
  6. After the operation is complete the signed flight logs from the drone can be uploaded to the server.
