Get started with Flight Permissions, generally the following is the flow for permission artefacts:

  • Create a Flight Plan (Upload Coordinates file as GeoJSON in Aerobridge)
  • Create a Flight Operation: Assign a flight plan to a operation and a drone, you will have to input additional fields around the operation.
  • Before you submit a request, you will need to create a "Permission Application", think of it as building a Application packet for requesting the permission artefact
  • You can then download a permission artefact or apply for one if you have done so.
  • After the operation is complete the signed flight logs from the drone can be uploaded to the server.
